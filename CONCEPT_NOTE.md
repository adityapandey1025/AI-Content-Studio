# Concept Note

## Project Name

AI Content Studio

## Problem Statement

Students, professionals, and small businesses often spend significant time drafting high-quality written content for different platforms. Manual writing can be slow, inconsistent, and difficult when users need multiple formats quickly.

## Objective

Build a full stack web application that generates high-quality content in multiple formats (blog, email, social posts, summaries, and professional documents) using a modern LLM API with secure backend streaming.

## Target Audience

- University students
- Job seekers
- Startup teams
- Content creators
- Marketing professionals
- Business analysts

## Key Features

- Multi-format AI content generation
- Real-time streaming output
- Markdown rendering
- Copy / TXT / PDF export
- Prompt templates
- Recent prompt history
- Dark/light mode
- Usage analytics on client side

## LLM Used

- OpenRouter
- Default model: `deepseek/deepseek-chat-v3-0324:free`

## Expected Outcome

A production-ready, secure, responsive AI writing studio deployable via Docker and AWS Elastic Beanstalk with a public URL.
